Перечислите и опишите семантические элементы HTML5 для текстового содержимого.

Элемент article. Определяет независимое, автономное содержимое, например, статью или пост в блоге.
Элемент aside. Определяет содержание в стороне от содержимого страницы.
Элемент details. Определяет дополнительные сведения, которые пользователь может просматривать или скрывать.

Перечислите и опишите элементы для поддержки аудио, видео и подключаемых модулей.
Элемент audio.Внедряет звуковой файл на веб-страницу.Путь к аудиофайлу указывается атрибутом src, а атрибут controls добавляет панель управления (кнопка запуска, прокрутка, регулятор громкости).
Элемент video. Встраивает цифровое видео на веб-страницу. Атрибут src указывает URL-адрес видеофайла, а атрибут controls добавляет элементы управления, с помощью которых пользователь сможет взаимодействовать с плеером
Элемент source. Позволяет добавлять файлы одновременно через тег с атрибутом src, что полезно, так как не все браузеры поддерживают все аудиоформаты.

Перечислите и опишите типы позиционирования, которые вы знаете.
Нормальное. Если для элемента не задано свойство position или ему задано значение static, слои позиционируются друг за другом, при этом каждый следующий начинается с новой строки.
Абсолютное. При задании абсолютного позиционирования элемента он «отделяется» от потока документа и перестаёт в нём существовать. Его местоположение, равно как и координаты, задаётся относительно краёв окна браузера.
Относительное. В случае задания свойству position значения relative позиционирование элемента будет установлено относительно его места по умолчанию.
Фиксированное. Если свойству position задать значение fixed, элемент будет привязан к определённой точками на экране и не будет изменять своего положения при прокрутке веб-страницы.

Позиционирование внутри элемента.
Для позиционирования внутри элемента можно использовать относительное позиционирование position: relative. 4 Оно позволяет настраивать положение дочернего элемента относительно себя, смещая его в разных направлениях относительно верхней, правой, нижней и левой границ родительского контейнера.

Перечислите и опишите составные компоненты формы.
Элемент form. Удерживает вместе все элементы управления формы и указывает браузеру, куда отправить данные после нажатия пользователем кнопки отправки.
Однострочное текстовое поле. Создаётся с помощью элемента input type="text" или input type="password". Если для атрибута type указано значение password, вводимые пользователем символы отображаются в виде звёздочек (*) или маркеров-точек (•).
Многострочное текстовое поле. Создаётся с помощью элемента textarea. В него можно ввести несколько строчек текста.
Флажок. Выводится элементом input type="checkbox". Флажок можно установить или очистить.
Переключатель. Выводится элементом Кнопка. Выводится элементами input type="submit", input type="image", input type="reset", input type="button". Кнопка типа submit собирает информацию с формы и отправляет её для обработки. Кнопка типа image делает то же самое, но позволяет вместо текста на кнопке вставить изображение. Кнопка типа reset очищает поля формы от введённых пользователем данных.
Список. Создаётся элементом select. Выводит список, из которого пользователь может выбирать значения.

Перечислите и опишите глобальные атрибуты тега в HTML5.
accesskey — определяет клавишу для быстрого доступа к элементу.
class — задает класс CSS, который будет применяться к элементу.
contenteditable — определяет, можно ли редактировать содержимое элемента.
contextmenu — определяет контекстное меню для элемента, которое отображается при нажатии на элемент правой кнопкой мыши.
dir — устанавливает направление текста в элементе.
